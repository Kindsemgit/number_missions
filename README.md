# number_missions
Приложение по расчету числа Миссии
Основные функции приложения:
Ввод даты рождения в формате ДД.ММ.ГГГГ с автоматической вставкой точек
Расчет числа миссии по алгоритму 
Подробное описание для каждого числа (1-9) и мастер-чисел (11, 22, 33)
Адаптивный дизайн, который работает на компьютерах и мобильных устройствах
Плавная анимация и приятный интерфейс
